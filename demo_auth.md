# Демонстрация системы авторизации

## Что реализовано:

1. ✅ **Страница входа** - `/login.html`
   - Красивый интерфейс с градиентом
   - Поля для логина и пароля
   - JavaScript для отправки запросов

2. ✅ **API эндпоинты** (частично):
   - `/api/v1/auth/login` - вход
   - `/api/v1/auth/me` - информация о пользователе
   - `/api/v1/auth/logout` - выход

3. ✅ **Модель User**:
   - Хеширование паролей через bcrypt
   - JWT токены для авторизации
   - Проверка прав доступа

## Как протестировать:

1. Откройте http://172.25.0.14/login.html
2. Введите логин: `admin`
3. Введите пароль: `admin123`
4. Нажмите "Войти"
